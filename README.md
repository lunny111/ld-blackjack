# ld-blackjack
Blackjack Script for FiveM (with UI)

## Dependencies
- ld-lib
- ld-ui

## How to Setup

1 - Start all scripts in your server.cfg

2 - add "casino_chips" item to your SQL (items table).

3 - Say "Thank you Lunny" when you are winning in blackjack.
